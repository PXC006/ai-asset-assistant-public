# AI复利资产助手｜公开体验版

这是 AI复利资产助手的公开体验版，用于让朋友或公开访客体验资产管理辅助分析流程。

## 公开版说明

- 所有个人数据已清零。
- 用户可自行填写示例数据进行体验。
- 公开体验版不建议填写敏感个人资产信息。
- 数据不保证长期保存。
- 本工具仅用于资产管理辅助分析，不构成投资建议。

## 数据库

公开版使用独立空数据库：

```text
data/public_empty.db
```

第一次运行时，程序会自动创建 `data/public_empty.db` 和所需表结构。公开版不会读取私人版的 `data/portfolio.db`。

## 启动方式

双击运行：

```text
start_public_app.bat
```

也可以在当前目录执行：

```powershell
streamlit run app.py
```

## 功能保留

公开体验版保留页面 UI、候选标的池、动态候选筛选器、基金 ETF 分析、代码识别、量化实验室、宽基方向选择助手、定投执行计划和再平衡规则。

## 隐私提醒

公开体验版不会内置真实资产、现金流、备用金、持仓、自选池、交易记录或历史月报。请勿提交 `.env`、`.streamlit/secrets.toml`、数据库文件或任何真实个人数据。
